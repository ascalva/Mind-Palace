### The loop, stated

Active sensor **orients attention** ‚Üí the agent **acts** (perturbs) ‚Üí the agent **senses the
reaction** ‚Üí the reaction **confirms** the action ‚Üí closes back to sensing. A closed sensorimotor
cycle; the owner names it a **causal loop**. This is the perception‚Äìaction cycle / active
inference `[FROM MEMORY ‚Äî Friston free-energy; verify before any book cites]`: act to test and
confirm your model; the mismatch drives what you learn.

### Orchestrator chew ‚Äî the loop lands on things we already built, and one thing it reveals

- **‚öë A causal loop IS a C-fiber structure ‚Äî so the agent WRITES its own causal graph by acting.**
  This is the payoff. sense‚Üíact is a causal edge; act‚Üíobserved-reaction is a *witnessed* causal
  edge (the integrator mints exactly these ‚Äî C = causal-witnessed production, `E_proven`). So the
  agent's sensorimotor loop literally inscribes **C-edges** into the corpus, and a closed loop is a
  **directed causal cycle ‚Äî precisely what the census/helix detector reads back** (bp-080; the
  helix CN-6). The agent's agency authors the C-fiber; the dreamer reads it as the agent's own
  causal history. Self-authored causality ‚Äî the ouroboros the founding note named, made mechanical.
- **"Confirmed by its senses" = PREDICTION/CONFIRMATION ‚Äî and we built the internal version.** The
  agent predicts the reaction, acts, senses the actual reaction, and confirms (low surprise) or
  diverges. Divergence is not failure ‚Äî it is the **prediction-residual = the creative/innovation
  signal** (the edge-dynamics-vector-field novel_object: "predict the resting-state so the owner's
  perturbation stands out"; the dreamer-as-forecaster capsule; and bp-082's influence diff, which
  is *exactly* "predict the graph's response to a perturbation, then measure it"). Confirmation vs
  surprise is the learning signal.
- **Two loops ‚Äî one CLOSED and live, one OPEN and gated (this is the honest status).**
  - *Internal loop (built, closed, LIVE):* dream/hypothesis ‚Üí perturb the GRAPH (the HYPOTHETICAL
    overlay) ‚Üí sense the response (the influence diff) ‚Üí confirm/condition (bp-079 + bp-082). No
    external gate ‚Äî it is a hypothetical read, so the agent already runs its full **cognitive**
    loop internally.
  - *External loop (built, OPEN at the gate, DORMANT):* sensing hands (orient) ‚Üí acting hands
    (perturb the world) ‚Üí observed stratum (sense the reaction). But **acting hands are gated to
    max tier NONE** (finding-0011; "the reasoner proposes, the effector executes, the gate makes it
    trustworthy" ‚Äî bright line 3). So the **embodied** loop is intentionally open at the *act*
    step: the agent may sense, orient, and predict, but acting on the world is human-gated and
    currently dormant. The confirmation loop is real today only for the internal graph, not the
    external world.
- **Reasoning and acting are the SAME object ‚Äî a typed causal path.** The fiber-chain grammar
  ruled a reasoning explanation IS a witnessed typed path (sense‚Üí‚Ä¶‚Üíconclude). The agent's action
  loop is also a typed causal path (sense‚Üíact‚Üíobserve). So thought and action share one
  representation ‚Äî the agent reasons in the same language it acts in, and the verb-licensing
  (C-move ‚Üí "caused/produced") governs both a reasoning claim and an action's witnessed effect.
  The agent's autobiography and its reasoning are written in one alphabet.
